arp - Предназначена для просмотра и изменения записей в кэш ARP, который представляет собой таблицу соответствия IP-адресов аппаратным адресам сетевых устройств.
arp -a - отобразить таблицу соответствия IP и MAC адресов для данного компьютера

hostname - отображает имя хоста

ping - отправить эхо-запрос по протоколу ICMP на имя или IP-адрес целевого узла
ping -l ... - Размер поля данных в байтах отправляемого запроса
ping -n ... - Число отправляемых эхо-запросов

nbtstat - используется для получения информации о подключениях с использованием протокола NETBIOS через TCP/IP

net - управление сетевой конфигурацией
net statistics - выводит журнал статистики для локальной службы рабочей станции или службы сервера
net user - используется для создания и изменения учетных записей пользователей на компьютерах

netstat - предназначена для получения сведений о состоянии сетевых соединений и слушаемых на данном компьютере портах TCP и UDP, а также 
для отображения статистических данных по сетевым интерфейсам и протоколам
netstat -a - Отображение всех подключений и ожидающих портов
netstat -b - Отображение исполняемого файла, участвующего в создании каждого подключения, или ожидающего порта

nslookup - предназначена для формирования запросов к серверам DNS

route - Обработка таблиц сетевых маршрутов
route print - печать маршрута

tracert - получить цепочку узлов, через которые проходит IP-пакет, адресованный конечному узлу

ipconfig - используется для отображения текущих настроек протокола TCP/IP и для обновления некоторых параметров, 
задаваемых при автоматическом конфигурировании сетевых интерфейсов при использовании протокола DHCP
